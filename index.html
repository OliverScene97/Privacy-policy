Политика конфиденциальности

1. Общие положения

1.1. Настоящая Политика конфиденциальности (далее — «Политика») действует в отношении всей информации, которую Администратор может получить о Пользователе мобильного приложения WHEELS City (далее — «Мобильное приложение» или «Сервис»).

1.2. Используя Мобильное приложение и предоставляя персональные данные, Пользователь подтверждает своё согласие на их обработку в соответствии с настоящей Политикой.

1.3. Администратор — команда разработчиков «WHEELS City dev team».

2. Персональные данные

2.1. Персональные данные — информация, относящаяся к определённому или определяемому Пользователю.

2.2. Обработка персональных данных включает сбор, запись, хранение, обновление, использование, передачу, блокирование и удаление.

2.3. Администратор обрабатывает следующие персональные данные Пользователя:

  • имя пользователя;
  • электронная почта;
  • номер мобильного телефона;
  • данные об автомобиле (марка, модель, VIN, фотографии);
  • геолокация Пользователя;
  • технические данные устройства: модель, ОС, IP, идентификаторы, данные о сбоях, push-токены и др.

2.4. Обрабатываются статистические данные с использованием аналитических систем (например, Yandex AppMetrica).

2.5. Для доступа к Сервису требуется регистрация (имя, email, телефон).

2.6. Регистрация считается фактом принятия настоящей Политики.

2.7. Пользователь обязан предоставлять достоверные данные.

2.8. После регистрации Пользователь получает уникальные учетные данные.

2.9. Пользователь несёт ответственность за безопасность своих учетных данных.

2.10. Пользователь соглашается на получение рекламных материалов.

2.11. Информирование о рекламных кампаниях возможно по любому предоставленному контакту.

3. Цели обработки данных

3.1. Обработка данных осуществляется для обеспечения работы Сервиса и предоставления функционала Мобильного приложения.

4. Порядок и условия обработки

4.1. Персональные данные обрабатываются без ограничения срока.

4.2. Обработка осуществляется на срок действия отношений между Пользователем и Администратором.

4.3. Все данные предоставляются пользователем либо его представителем.

4.4. Администратор вправе передавать данные государственным органам согласно законодательству.

4.5. Правовые основания обработки: Конституция РФ, ГК РФ, ТК РФ, согласие пользователя, договоры.

4.6. Для защиты данных применяются следующие меры:

  • защищённые каналы передачи (HTTPS, TLS);
  • хранение данных на серверах Supabase с доступом по ролям;
  • аутентификация по токенам;
  • ограничение доступа только уполномоченным лицам;
  • минимизация собираемых данных;
  • SSH-доступ к серверной инфраструктуре с использованием ключей;
  • отсутствие хранения данных на физических носителях.

5. Права пользователя

5.1. Пользователь вправе:

  • уточнять и обновлять свои данные;
  • требовать их блокирования или уничтожения;
  • запрашивать информацию об обработке своих данных;
  • получать сведения о сроках хранения и источниках данных.

6. Права и обязанности администратора

6.1. Администратор использует данные только для целей, описанных в Политике.

6.2. Администратор принимает разумные меры для защиты данных.

6.3. Срок хранения данных соответствует целям обработки и требованиям закона.

6.4. Администратор вправе не удалять данные, которые обязателен хранить по закону.
